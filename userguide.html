Create Project

1. log into account

2. go to "project view" of dashboard

3. Click red "Create New +" button

4. Add box number, member, and title to project




Download Data

1. View project from which you want to download the data

2. click the "download" button below member names

3. data will download into Excel file with .slxs extension




Find Existing Data

1. Go to "map view" dashboard

2. search in the left-hand search bar for a field

3. browse published data sets from box owners
